# AI Study Assistant Chatbot - User Guide

## Overview
The AI Study Assistant is a Gemini-powered chatbot that helps students with their educational needs. It's accessible only to students and provides intelligent assistance for studying, document analysis, and practice question generation.

## Features

### 1. **Conversational AI Assistant**
- Ask any educational questions
- Get detailed explanations on topics
- Receive study tips and guidance
- Chat history is automatically saved and persists across sessions

### 2. **File Upload & Analysis**
- Upload images, PDFs, or text files (up to 10MB)
- Supported formats: JPEG, PNG, GIF, WebP, PDF, TXT
- AI analyzes the content and can answer questions about it

### 3. **Document Summarization**
- Upload any educational document
- Click the summarize button to get a comprehensive summary
- Perfect for quickly understanding long documents or study materials

### 4. **Practice Question Generation**
- Upload study materials or notes
- AI generates 5-10 practice questions with answers
- Great for exam preparation and self-assessment

### 5. **Draggable Interface**
- Floating button on the right side of the screen
- Drag the button up or down to position it where you prefer
- Minimize/maximize the chat window as needed
- Clear chat history anytime

## How to Use

### Starting a Conversation
1. Look for the floating chat button on the right side of any student page
2. Click the button to open the chat interface
3. Type your question in the input field
4. Press Enter or click the Send button

### Uploading Files
1. Click the paperclip icon in the chat input area
2. Select an image, PDF, or text file from your device
3. Once uploaded, you can:
   - **Send with a message**: Type a question about the file and send
   - **Summarize**: Click the document icon to get a summary
   - **Generate Questions**: Click the question mark icon to create practice questions

### Managing Chat
- **Minimize**: Click the minimize icon to collapse the chat window
- **Clear History**: Click the trash icon to delete all chat messages
- **Close**: Click the X icon to close the chat window
- **Reposition**: Hover over the button and drag the grip handle to move it up or down

## Tips for Best Results

1. **Be Specific**: Ask clear, specific questions for better answers
2. **Provide Context**: Include relevant details about your subject or topic
3. **Use File Upload**: Upload study materials for more accurate, context-aware responses
4. **Break Down Complex Topics**: Ask follow-up questions to dive deeper into subjects
5. **Practice Questions**: Use the question generation feature regularly for exam prep

## Privacy & Data

- Chat history is stored securely in Firebase Firestore
- Only you can access your chat history
- Each student's chat data is completely private
- Clear your history anytime using the trash icon

## Technical Details

- **AI Model**: Google Gemini 1.5 Flash
- **Max File Size**: 10MB
- **Supported File Types**: Images (JPEG, PNG, GIF, WebP), PDF, TXT
- **Response Time**: Typically 2-5 seconds
- **Chat History**: Automatically saved after each message

## Troubleshooting

### Chat not responding?
- Check your internet connection
- Try refreshing the page
- Clear chat history and start a new conversation

### File upload failed?
- Ensure file is under 10MB
- Check that file format is supported
- Try a different file or compress large files

### Can't see the chat button?
- Make sure you're logged in as a student
- The button only appears on student pages
- Try refreshing the page
