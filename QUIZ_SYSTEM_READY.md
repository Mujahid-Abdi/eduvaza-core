## How to Use

### Creating a Quiz

1. **Login** as Teacher, School Admin, or Super Admin
2. Navigate to **Quiz Page** (from sidebar)
3. Click **"Create Quiz"** button
4. Fill in quiz details:
   - Title and Description
   - Quiz Type (Practice or Scheduled)
   - Language (English, French, Arabic, Swahili)
   - Difficulty (Easy, Medium, Hard)
   - Time Limit (optional)

5. Add questions:
   - Multiple Choice Questions (MCQ)
   - True/False Questions
   - Short Answer Questions
   - Set points and time limit per question

6. Configure settings:
   - **Publish Quiz** toggle (ON = visible to public, OFF = draft)
   - Shuffle Questions
   - Shuffle Options
   - Show Results
   - Passing Score

7. Click **"Save Quiz"**

### Viewing Quizzes

**As a Creator (Teacher/School/Admin):**
- Go to your Quiz page
- See tabs: All, Published, Drafts, Scheduled, Completed
- Manage your quizzes (edit, schedule, view analytics)

**As a Student/Public:**
- Go to `/quizzes` page
- See all published quizzes
- Filter by difficulty, language
- Search by title/description

**As Super Admin:**
- Go to "Manage Quizzes"
- See ALL quizzes from all users
- View statistics
- Delete any quiz

## Data Structure

Each quiz in Firebase contains:
```
{
  title: "Quiz Title"
  description: "Quiz description"
  quizType: "practice" | "scheduled"
  teacherId: "creator-uid"
  teacherName: "Creator Name"
  language: "en" | "fr" | "ar" | "sw"
  difficulty: "easy" | "medium" | "hard"
  questions: [...] // Array of questions
  totalPoints: 100
  timeLimit: 15 // minutes
  isPublished: true | false
  shuffleQuestions: true | false
  shuffleOptions: true | false
  showResults: true | false
  passingScore: 60 // percentage
  createdAt: timestamp
  updatedAt: timestamp
}
```
