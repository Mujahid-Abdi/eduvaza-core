## ðŸš€ How to Use

### For Students

1. **Access the Chatbot**
   - Log in as a student
   - Look for the floating chat button on the right side of any student page
   - Click the button to open the chat interface

2. **Ask Questions**
   - Type your question in the input field
   - Press Enter or click Send
   - Wait for the AI response (2-5 seconds)

3. **Upload Files**
   - Click the paperclip icon
   - Select a file (image, PDF, or text)
   - Choose an action:
     - **Send with message**: Ask a specific question about the file
     - **Summarize**: Get an automatic summary
     - **Generate Questions**: Create practice questions

4. **Manage Chat**
   - **Drag**: Hover and drag the grip handle to move the button up/down
   - **Minimize**: Click the minimize icon to collapse the chat
   - **Clear History**: Click the trash icon to delete all messages
   - **Close**: Click the X to close the chat window
