TROUBLESHOOTING
---------------

Problem: Still seeing login page
Solution: 
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Hard refresh (Ctrl+F5)
  3. Close all browser tabs
  4. Restart dev server
  5. Open in incognito window

Problem: Cannot access courses
Solution:
  1. Check URL is /courses (not /course)
  2. Check dev server is running
  3. Check console for errors (F12)

Problem: Enroll button not working
Solution:
  1. This is expected for guests
  2. Should show "Login Required" message
  3. Login first, then try again
